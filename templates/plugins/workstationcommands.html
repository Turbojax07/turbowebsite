{% extends 'base/inline_md.html' %} {% block title %}WorkstationCommands{% endblock %} {% block header %}WorkstationCommands{% endblock %} {% block md %} # WorkstationCommands This is my first published minecraft plugin! It adds commands for each workstation in minecraft. These commands provide as much configuration as I can reasonably give. You can enable/disable commands, add/remove aliases, and manage specific permissions.
